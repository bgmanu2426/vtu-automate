Before You Launch

  • Use your online.vtu.ac.in login, not university portal credentials.
  • Course slug is extracted automatically if you paste the full learning URL.
  • You will get a Job ID - save it to reconnect if you close the tab.
  • Your password is never stored and is wiped after completion.
  • Jobs run in queue order when traffic is high.

Run It On Your Own Machine

Three commands to run locally with your own queue and your own credentials.

1) git clone https://github.com/vikas-bhat-d/vtu-course-automation && cd vtu-course-automation && npm install
2) npm run serve
3) open http://localhost:3000
